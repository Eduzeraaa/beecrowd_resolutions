# Repositório de Resoluções Beecrowd

Este repositório contém minhas soluções pessoais das questões do Beecrowd em Python. 
O objetivo principal é **prática de lógica, algoritmos e programação competitiva**, 
bem como registrar minha evolução.

---

## Como usar

1. Cada arquivo contém uma solução de questão individual.
2. O enunciado das questões pode ser encontrado no site oficial do Beecrowd: [https://www.beecrowd.com.br/judge/pt](https://www.beecrowd.com.br/judge/pt)
3. Os arquivos estão escritos em Python, usando listas, loops e funções básicas.
4. Recomenda-se estudar a lógica do código antes de rodar para entender cada solução.

---

## Observações

- Este repositório é **pessoal** e serve como histórico de aprendizado.
- Não inclui os enunciados completos por questões de copyright.
- O objetivo principal é **fortalecer minha base de programação**, melhorar a lógica e a disciplina de resolução de problemas.
